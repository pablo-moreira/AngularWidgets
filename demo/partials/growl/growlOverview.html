The Mac's growl info like messages.
  • Show a skinnable message on the right top for a certain time.
  • Messages can be made sticky
  • Icon images must be specified in application
			.pui-growl-image {
			    background: url("../demo/images/messages.png") no-repeat;
			}
			
		

		
		

		
			puiGrowl is a service that can be used to show 'growl' messages. 

            showInfoMessage = function(title, mesg) 

            showWarnMessage = function(title, mesg) 

            showErrorMessage = function(title, mesg) 

            clear = function() -> remove messages

            setSticky = function(sticky) -> Keep messages

            setLife = function(time) -> How long are the messages visible (milliseconds, default 3000)
