Grouping component which also provides content toggling. Supports
  • Grouping of content skinned according to the theme
  • Optional toggleable content
  • Expression allowed in legend tag and within body
  • Programmatic toggleable
  • Callback function available when user toggles the content
  • No warning when callback set but not the collapsed option.
Name Type Default Description
toggleable Boolean false Will there be a + and - to toggle the contents of the fieldSet.
collapsed Boolean undefined When set, defines the display status of the fieldset.
onStateChanged function() undefined Allows to react when the user changes the collapsed state of the content.
