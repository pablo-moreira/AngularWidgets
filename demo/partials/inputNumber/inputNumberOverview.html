Extension to standard inputText with skinning capabilities.
  • Support for 'rendered', showing and hiding field.
  • Support for disabled.
  • Showing/hiding and enable/disable field can be animated.
Name Type Default Description
value String undefined Link from field content to scope variable.
type String Text The type of the input field. By default text is assumed.
pui-disabled String undefined Link to scope variable which defines if the field is disabled or not.
rendered String undefined Link to scope variable which defines if the field is visible.
