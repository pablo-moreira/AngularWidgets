Datatable displays data in tabular format. Pagination, Sorting, Selection are built-in features supporting both local and remote data.
  • Themeable table representation of data
  • Takes javascript array or function to supply data
  • Autodetection of columns based on properties of first object in array
  • Callback executed when row selected
  • Columns (properties of object) to show can be defined with nested pui-column directive
  • Sortable columns
  • Pagination of table
  • Table caption and column headers definable
  • Multiple row selection
  • Programmatic setting selected row(s) and page
Name Type Default Description
value Array of objects / function(callback) null Data to shown as an array of objects or function that supplies (remotely) the info.
selectionMode String null Determines if only one (value single) or multiple rows can be selected.
rowKey String null Name of the property used to uniquely identify the row.
onRowSelect function(event, data) null When specified, it activates the selection possibilities and function is called when user selects a row in the table.
onRowUnselect function(event, data) null When specified, it is called when user deselects a row in the table. No effect when selection options is not activated ('rowUnselect' or 'rowUnselectAll' specified as event)
onPage function(event, data) null When specified, it is called when user changes the page (paginated data)
binding Object undefined Retrieve the options from a JavaScript object.
tableData (only in JavaScript object) Array of objects / function(callback) undefined When the binding attribute is used, the data to show is retrieved from this property.
paginator boolean false Is the data paginated.
rows int false The number of rows on each page of data.
caption String undefined Table header name.

column definition

Column information isn't needed but when specified, this is the where the information is taken from.
  1. Child elements with pui-column directive on
  2. Autodetect from the first object in the data

pui-column directive

Name Type Default Description
value String null Name of the property.
sortable boolean false Is this column sortable.
headerText String undefined Column header value.
