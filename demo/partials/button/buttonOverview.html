Button provides styling over native button elements.
  • Themeable buttons
  • Button can be disabled
  • Support for 'rendered', showing and hiding field.
  • Support for icons
  • Dynamic value using Angular Expressions
  • Showing/hiding and enable/disable field can be animated.
Name Type Default Description
value String null text displayed on the button.
actionListener String -> function() null Method linked to the controller which is executed when clicked on the button.
pui-disabled String undefined Link to scope variable which defines if the field is disabled or not.
icon String null Defines the icon of the button. (from theme)
iconPos String left Position of the icon, valid values are "left" and "right".
