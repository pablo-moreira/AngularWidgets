Button provides styling over native button elements.
  • Themeable buttons
  • Button can be disabled
  • Support for 'visible', showing and hiding field.
  • Support for icons
  • Dynamic value using Angular Expressions
  • Showing/hiding and enable/disable field can be animated.
Name Type Default Description
value String null Text displayed on the button.
action String -> function() null Method linked to the controller which is executed when clicked on the button.
disabled boolean false Link to scope variable which defines if the field is disabled or not.
icon String null Defines the icon of the button. (from font awesome)
iconPosition String left Position of the icon, valid values are "left" and "right".
visible boolean left Link to scope variable which defines if the field is visible or not.
