Grouping component providing with content toggle and close features.
  • Toggling and closing can be specified in the view or in the controller
  • Title attribute supports AngularJS expressions
  • Collapsing and expanding panel can be triggered programmatic
  • Toggle orientation can be configured.
  • Showing and hiding panel content can be animated.
  • Feedback on user actions (toggling and closing)
  • Support for 'rendered' (showing and hiding panel)
Name Type Default Description
header String undefined Panel title
toggleable Boolean false Can the panel be collapsed and expanded.
collapsed Boolean false Initial state of the panel contents.
closeable Boolean false Can the panel be removed.
toggleOrientation String vertical Defines orientation of toggle direction, valid values are "vertical" and "horizontal".
onStateChange String -> function () undefined Linked to a function in the scope which is called when user expand/collapse a panel.
onClose String -> function () undefined Linked to a function in the scope which is called when user closes a panel.
binding Object undefined Retrieve the options from a JavaScript object.
