Tabbed panel component featuring client side tabs
  • Themeable panel components
  • Title can be on top or on the left, also definable by angular expressions
  • Tabs can be removed
  • Programmatic setting the active tab
  • User callback when tab is selected
  • Panel can be in an included file.
  • Support for changing number of tabs dynamically
  • puiPanel can be used to define the panel
  • Support of effects
  • Lazy loading
Name Type Default Description
closeable Boolean false Can the tab panels be closed?
orientation String top Where are the tab panels located? On the 'top' or on the 'left'?
onTabChange String -> function (index) undefined Linked to a function in the scope which is called when user selects another tab.
binding Object undefined Retrieve the options from a JavaScript object.
On the object bound to the widget with the binding attribute, an additional function is created setActiveTab = function(idx) which can be used to change the active tab progmatically.
